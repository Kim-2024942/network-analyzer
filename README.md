# Network Traffic Analyzer (Cybersecurity Packet Monitoring Tool)

##  Project Overview
The **Network Traffic Analyzer** is a Python-based cybersecurity tool developed using **Scapy** in a **Kali Linux environment**.  
It is designed to capture and analyze live network packets in real time to help understand network behavior and detect potential suspicious activity.

This project demonstrates basic principles of **network traffic monitoring and cybersecurity analysis**.

---

##  Features
-  Live packet capture from network interfaces
-  Traffic analysis (TCP and UDP packet identification)
-  Monitoring of source and destination IP addresses and ports
-  Basic detection of unusual or suspicious network activity patterns

---

##  Tools & Technologies Used
- Python 3
- Scapy (packet manipulation and sniffing library)
- Kali Linux (testing environment)
